candideShapeproperty
candideShape:BRFCandideShape  [read-only]

Result data when: vars.mode == BRFMode.FACE_TRACKING && state == BRFState.FACE_TRACKING && vars.faceTrackingVars.extensionCandideEnabled == true. Includes a transformed Candide 3 model with translationX, translationY, scale, rotationX, rotationY and rotationZ of the face, also its vertices and trianlges. This Candide 3 model is used to update 3D engines.

Same as BeyondRealityFaceManager.candideShape

faceShapeproperty 
faceShape:BRFFaceShape  [read-only]

Result data when: vars.mode == BRFMode.FACE_TRACKING && (state == BRFState.FACE_TRACKING_START || state == BRFState.FACE_TRACKING). Includes 68 points/vertices of the face shape, also the triangles and the shape bounds.

ExampleWebcamFaceEstimation_Output.jpg shows all indices of the points.

Same as BeyondRealityFaceManager.faceShape

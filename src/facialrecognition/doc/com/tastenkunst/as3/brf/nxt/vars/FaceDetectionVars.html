levelproperty 
level:int

Sets/Gets the pyramid level FACE_DECTION is working on. Keep in mind, that level 0 is the source image size, level 1 is half that size, level 2 is size / 4 and level 3 is size / 8. You will have to divide all other params by 2 / 4 / 8, if you set a level other than 0.

e.g.: level 0: baseScale 4.0 or 5.0 - level 1: baseScale: 2.5

default: 0
